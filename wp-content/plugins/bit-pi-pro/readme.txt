== API Usage ==

Bit Pi is a powerful integration tool that uses platform API to send and receive data with users' permission.
some operations are done in the frontend(Javascript) and some are done in the backend(PHP). Bit Pi did not collect any integration data from users.
But collect errors, logs, and usage data to improve the plugin.
currently, we are using the following platforms API.

***[Google Sheets API](https://www.google.com/sheets/about/)***
We are using the REST APIs to interact programmatically with Google Sheets. [View documentation
](https://developers.google.com/sheets/api)

*Google API Services [Terms](https://developers.google.com/terms) & [Policy](https://developers.google.com/terms/api-services-user-data-policy)*

**[Google Drive API](https://www.google.com/drive/)**
We are using the REST APIs to interact programmatically with Google Drive. [View documentation
](https://developers.google.com/drive/api)

*Google API Services [Terms](https://developers.google.com/terms) & [Policy](https://developers.google.com/terms/api-services-user-data-policy)*

**[Google Calendar API](https://workspace.google.com/products/calendar/)**
We are using the REST APIs to interact programmatically with Google Calendar. [View documentation
](https://developers.google.com/calendar/api)

*Google API Services [Terms](https://developers.google.com/terms) & [Policy](https://developers.google.com/terms/api-services-user-data-policy)*

**[Jot Form API](https://fluentforms.com/)**
The Jotform API is written for Jotform users and 3rd party developers who would like to serve Jotform users. [View documentation](https://api.jotform.com/docs/)

*Jot Form [Terms](https://www.jotform.com/terms/) & [Policy](https://www.jotform.com/privacy/)*

**[Zoho CRM API](https://www.zoho.com/crm/)**
Integrate the Zoho CRM modules with any third-party application. [View documentation](https://www.zoho.com/crm/developer/docs/api/v2)

*Zoho CRM API [Terms](https://www.zoho.com/crm/terms.html) & [Policy](https://www.zoho.com/privacy.html)*

== Installation ==

1. Download the plugin.
2. From the WordPress Admin Panel, click on Plugins => Add New.
3. Click on Upload, so you can directly upload your plugin zip file.
4. Use the browse button to select the downloaded plugin zip file, and then click Install Now.
